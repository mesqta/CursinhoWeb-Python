```python
Operador: * (Multiplicação)
Operador: / (Divisão)
Operador: % (Módulo/Resto da Divisão)
Operador: ** (Potência)
Operador: // (Divisão Inteira)
Operador: >> (Deslocamento para a Direita)
Operador: << (Deslocamento para a Esquerda)

<  (Menor que)
<= (Menor ou Igual a)
>  (Maior que)
>= (Maior ou Igual a)
!= (Diferença)
```
